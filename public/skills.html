# fsdb_analyzer.py - Quick Reference Guide

## ⚠️ CRITICAL: Tool Has TWO MODES - Never Mix Them! ⚠️

## MODE 1: FSDB Mode (--file)
**Parameter:** `--file <fsdb_path>`
**Use for:** Reading signal values, searching hierarchy

| Operation | Purpose |
|-----------|---------|
| `--get-value` | Get signal value at specific time |
| `--search-signal-value` | Find when signal had value |
| `--search-signals` | Search for signals in scope |
| `--search-scopes` | Search for modules/hierarchy |
| `--get-signal-info` | Get signal metadata |

## MODE 2: Database Mode (--dbdir)
**Parameter:** `--dbdir <database_dir>`
**Use for:** ONLY tracing signal drivers

| Operation | Purpose |
| `--trace-drivers` | Find what drives a signal |

## ❌ COMMON MISTAKES - DO NOT DO THESE!

1. ❌ Mixing modes:
  # WRONG!
  fsdb_analyzer.py --file sim.fsdb --dbdir simv.daidir

2. ❌ Using `python` command:
  # WRONG!
  python dv/tools/fsdb_analyzer.py ...
  Correct: dv/tools/fsdb_analyzer.py ...

3. ❌ Using `--fsdb` instead of `--file`
4. ❌ Using `--dbdir` for value checking

How it works

Skills = your docs, made executable

A skill is a SKILL.md file — plain markdown that gives Claude domain knowledge and step-by-step procedures for a specific task.

  • Your reference docs, structured so Claude can act on them — not just display them
  • Activate with a slash command: /dv-reference, /dma-agent
  • Shared across the team via git — everyone gets the same expertise
  • auto_debug.md is already 90% of a skill — rules, tools, procedures. Skills just make it executable.
# .claude/skills/dv-reference/SKILL.md
---
name: dv-reference
description: CASM directed test writing
 and DMA debugging reference
---

## Rules
1. Sequential logic → check 1ps before
2. Combinatorial → check at same time
3. Clock edge → value 1ps before edge
4. Trace signals to fabric, not testbench
5. Search register config from end of log

## Tools
dv/tools/fsdb_analyzer.py
 --dbdir mode for driver tracing
 --file mode for everything else
 NEVER mix modes

## Design Docs
fabric.md — RTL routing & signals
fab_adder.md — Color 10/11 adder
dma_agent_guide.md — DMA descriptors
